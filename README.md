# 🏨 Hotel Housekeeping Management System – Business Analysis Case Study

## 📌 Project Overview

This project demonstrates an end-to-end Business Analysis case study for a Hotel Housekeeping Management System developed using Agile methodologies. The solution aims to improve housekeeping operations by enabling efficient staff scheduling, room allocation, cleaning task management, and progress tracking across multiple hotel locations.

## 🎯 Business Objective

To streamline hotel housekeeping operations through a mobile-based application that enables:

* Staff availability management
* Room assignment and allocation
* Cleaning schedule management
* Cleaning progress tracking
* Supervisor reviews and ratings
* Operational reporting and monitoring

## 👥 Stakeholders

* **Cleaning Staff** – Complete assigned cleaning tasks and update progress.
* **Supervisor** – Monitor cleaning activities, verify work quality, and provide ratings.
* **Admin/Manager** – Manage staff allocation, schedules, room assignments, and reporting.
